Molecule#

class pyscf.gto#
class mole#
Parameters

  • verbose (int) – Print level

  • spin (int) – num. alpha electrons - num. beta electrons to control multiplicity. If spin = None is set, multiplicity will be guessed based on the neutral molecule.

  • atom (List) – To define the molecular structure. Example:

>>> from pyscf import gto
>>> mol = gto.M(atom='H 0 0 0; F 0 0 1', basis='6-31g')
>>> mol.atom
>>> mol.elements
>>> 'H 0 0 0; F 0 0 1'
